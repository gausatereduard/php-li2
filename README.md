# Finance Manager

Веб-приложение для управления личными финансами. Позволяет отслеживать доходы и расходы, управлять кошельками, категориями и бюджетом.

## Возможности

- **Управление кошельками** - создание и редактирование кошельков с разными валютами
- **Транзакции** - добавление доходов, расходов и переводов между кошельками
- **Категории** - личные и глобальные категории для транзакций
- **Бюджеты** - установка лимитов расходов по категориям
- **Курсы валют** - отображение курсов MDL к USD и EUR (обновляется раз в сутки)
- **Панель администратора** - управление пользователями

## Технологии

- PHP 8.x
- PostgreSQL
- Vanilla PHP MVC
- HTML/CSS (без фреймворков)

## Запуск с Docker Compose

```bash
docker-compose up --build
```

Приложение будет доступно по адресу: http://localhost:8000

## Настройка базы данных

Миграции выполняются автоматически при первом запуске. Основные таблицы:

- `users` - пользователи
- `wallets` - кошельки
- `categories` - категории
- `transactions` - транзакции
- `budgets` - бюджеты
- `exchange_rates` - курсы валют
